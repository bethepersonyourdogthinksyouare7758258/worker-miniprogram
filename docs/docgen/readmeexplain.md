# 劳动仲裁申请书生成工具 - 通俗易懂说明

## 🎯 这个网站是做什么的？

这是一个帮助打工人自动生成劳动仲裁申请书的网站。就像填表格一样，你只需要填写自己的基本信息、公司信息、工资情况等，系统就会自动帮你生成一份标准格式的仲裁申请书，可以直接用来申请劳动仲裁。

## 🏗️ 网站是怎么实现的？

### 简单原理
想象一下这个网站就像一个智能的文档生成器：

1. **前台（用户看到的页面）**：就像银行的取号机屏幕，用户在这里填写信息
2. **后台（处理数据的地方）**：就像银行柜台后面的工作人员，负责处理用户提交的信息
3. **数据库（存储信息的地方）**：就像银行的档案柜，存放所有的记录
4. **模板（文档格式）**：就像标准的申请表格式，往里面填入具体信息

### 工作流程
```
用户填写表单 → 提交到后台 → 后台处理数据 → 套用模板 → 生成文档 → 返回给用户
```

## 🔄 系统工作流程

1. **用户访问网站**
   - 浏览器加载前端页面（HTML + CSS + JavaScript）
   - 显示表单界面

2. **用户填写信息**
   - 在表单中输入个人信息、公司信息、争议信息等
   - 前端进行基本验证（比如检查身份证格式）

3. **提交数据**
   - 前端通过API接口把数据发送给后端
   - 就像把填好的表格交给工作人员

4. **后端处理**
   - 后端接收数据并验证
   - 将数据存储到数据库
   - 调用文档生成服务

5. **生成文档**
   - 系统根据预设模板
   - 把用户信息填入模板的相应位置
   - 自动计算加班费、赔偿金等

6. **返回结果**
   - 生成的文档返回给前端
   - 用户可以预览、下载或复制文档内容

XXX（公司名）：
因下列原因，根据《劳动合同法》第三十八条的规定

（一） 未按照劳动合同约定提供劳动保护或者劳动条件的；
（二） 未及时足额支付劳动报酬的；
（三） 未依法为劳动者缴纳社会保险费的；
（四） 用人单位的规章制度违反法律、法规的规定，损害劳动者权益的；
（五） 因本法第二十六条第一款规定的情形致使劳动合同无效的；
（六） 法律、行政法规规定劳动者可以解除劳动合同的其他情形。
